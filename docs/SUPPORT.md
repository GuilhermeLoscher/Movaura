# Movaura support

For Microsoft Store listing, configure the official support channel before submission.

Suggested support information:

- Product: Movaura
- Developer: Guilherme Loscher
- Support contact: define official e-mail or website before Store submission

When requesting support, include:

- Windows version;
- Movaura version;
- monitor count and DPI;
- wallpaper type;
- whether the app was installed from MSIX or standalone;
- support report generated inside the app, if requested.

Do not send private beta keys, passwords, PFX files or screenshots containing secrets.
