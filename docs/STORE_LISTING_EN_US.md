# Movaura - Microsoft Store listing en-US

## Name

Movaura

## Short description

Live wallpapers for Windows with videos, images, GIFs, effects and performance profiles.

## Long description

Movaura turns videos, images, GIFs and effect scenes into live wallpapers for Windows. It is designed to be approachable for beginners, with a clear home panel, local library, preview, quick apply flow and performance profiles.

Main features:

- apply videos, images and GIFs as wallpapers;
- local library with file import;
- Light, Recommended and Maximum quality profiles;
- multi-monitor support;
- local mock AI creation flow for architecture demonstration;
- isolated native compositor;
- pause, stop and restore controls;
- optional FFmpeg video optimization when available.

Note: online features, beta activation and real AI providers depend on the build and distributor configuration.

## Suggested category

Personalization.

## Functional notice

Movaura uses Win32 features and a native compositor to present animated wallpapers on the desktop. In some configurations, it can use an isolated surface when Explorer does not allow direct rendering behind icons.
